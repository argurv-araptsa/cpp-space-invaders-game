# Space Invaders – Python Game

A 2D Space Invaders-style arcade game developed in Python using the PyGame library.

This project was created as part of academic training in Python programming and game development at IEK OMIROS, focusing on real-time systems and interactive application design.

---

## Features

- Player movement (left / right controls)
- Enemy spawning system
- Bullet shooting mechanics
- Collision detection (bullets vs enemies)
- Score tracking system
- Game over logic
- Increasing difficulty over time

## Technologies Used

- Python 3.x
- PyGame library

---

## Project Purpose

This project was developed to strengthen understanding of:
- Game loop architecture
- Event-driven programming
- Collision detection systems
- Real-time application logic
- Object interaction in a graphical environment
